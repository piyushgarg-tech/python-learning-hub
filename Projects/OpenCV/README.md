# Assignment 11 - OpenCV Basics

## Project Overview

This project is made using Python and OpenCV.

In this assignment, I learned how to work with images and videos using OpenCV. I practiced different image processing operations such as resizing, flipping, rotating, thresholding, edge detection, filters, and video recording.

The purpose of this project was to understand the basic features of OpenCV and how computer vision works.

---

## Features

### Image Reading and Resizing

* Read an image from a file
* Display image on screen
* Resize image
* Save resized image

### Morphological Operations

* Erosion
* Dilation
* Opening
* Closing
* Gradient
* Top Hat
* Black Hat

### Image Flipping

* Vertical Flip
* Horizontal Flip
* Both Side Flip

### Image Transformation

* Shift image
* Rotate image

### Threshold and Edge Detection

* Binary Threshold
* Canny Edge Detection

### Blur and Filters

* Gaussian Blur
* Median Blur
* Bilateral Filter

### Video Processing

* Open webcam
* Record video
* Save video
* Play recorded video
