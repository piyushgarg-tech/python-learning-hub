# Selenium Assignment

This is my Selenium assignment.

In this project I used Selenium with Python to automate some browser tasks.

First the program opens Google and searches for Selenium. After that it goes back and opens Amazon website. Then it opens Today's Deals page and searches for Gaming Laptop. At last it collects the product names and prints them in terminal.

## Things Used

* Python
* Selenium
* Chrome Browser
* ChromeDriver

## What Program Does

* Open Google
* Search Selenium
* Go back
* Open Amazon
* Open Today's Deals
* Search Gaming Laptop
* Get product names
* Print total products found

## How To Run

Install Selenium:

```bash
pip install selenium
```

Run the file:

```bash
python assignment.py
```

The program prints the total number of products found and also prints product names in terminal.
