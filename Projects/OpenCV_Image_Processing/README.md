# OpenCV Image Processing and Computer Vision

## Overview

This project was developed as part of my OpenCV learning journey. The objective was to gain hands-on experience with image processing, computer vision fundamentals, and webcam-based video handling using Python and OpenCV.

The project demonstrates several common computer vision operations including image resizing, flipping, filtering, edge detection, image transformations, morphological operations, and video recording.

## Features

### Image Resizing

* Load and display images
* Resize images while maintaining aspect ratio
* Save processed images

### Image Flipping

* Vertical flip
* Horizontal flip
* Combined horizontal and vertical flip

### Image Blurring

* Gaussian Blur
* Median Blur
* Bilateral Filtering

### Edge Detection

* Binary Thresholding
* Canny Edge Detection

### Image Transformations

* Image Translation (Shifting)
* Image Rotation

### Morphological Operations

* Erosion
* Dilation
* Opening
* Closing
* Gradient
* Top Hat
* Black Hat

### Webcam Video Capture

* Live webcam feed
* Video recording
* Playback of recorded video
* Custom frame overlays and text rendering

## Technologies Used

* Python
* OpenCV
* NumPy
