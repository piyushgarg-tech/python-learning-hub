# Browser Automation with Selenium

## Overview

This project demonstrates browser automation using Selenium WebDriver and Python. The application automates common web browsing tasks such as searching on Google, navigating web pages, interacting with Amazon, and extracting product information.

The project was built to gain hands-on experience with web automation, browser control, and element interaction using Selenium.

---

## Features

* Launch and control a web browser
* Automate Google searches
* Navigate between web pages
* Refresh pages programmatically
* Interact with buttons and links
* Search products on Amazon
* Extract product information from search results
* Automate repetitive browser tasks

## Selenium Concepts Demonstrated

### Browser Control

* Open browser
* Navigate to URLs
* Maximize browser window
* Close browser sessions

### Element Location Strategies

* By.NAME
* By.CLASS_NAME
* By.LINK_TEXT
* By.XPATH

### User Interaction

* Enter text into input fields
* Submit searches
* Click buttons and links
* Navigate back
* Refresh pages

### Data Extraction

* Locate multiple elements
* Extract product titles
* Count search results
* Display extracted information
