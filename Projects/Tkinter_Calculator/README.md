# Tkinter Calculator

A simple calculator application built using Python and Tkinter.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Percentage Calculation
- Decimal Number Support
- Basic Error Handling

## Concepts Practiced

- Tkinter GUI Development
- Entry Widget
- Button Widget
- Functions
- Lambda Functions
- Global Variables
- Exception Handling

## Purpose

This project was created to practice the fundamentals of GUI programming with Tkinter and understand how user input, widgets, and functions work together in a desktop application.

## Future Improvements

- Backspace Button
- Keyboard Support
- Calculation History
- Improved User Interface
