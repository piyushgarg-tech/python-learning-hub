# Assignment 05 - Data Structures and Strings

## Overview

This assignment demonstrates:

* Dictionaries
* User input
* List slicing
* List manipulation

---

## Task 1: Student Marks Dictionary

Creates a dictionary of student names and marks.

The program:

* Accepts a student's name
* Retrieves the corresponding marks
* Displays an error message if the student is not found

File:

task1_student_marks.py
